There are mainly 3 types of css

  • Inline css

  • This is an example of inline css.In inline css we are giving styles within the tag itself

  • Internal css

  • This is an example of internal css. Here we give styles within the head of the html also link them by class attribute and also by using "." this to identify the internal css.

  • External css

  • This is an example of external css. Here we style by giving the required styles for the document to another css sheet which is external to the current sheet and link them using the link tag using the href attribute.
